-모델 구성 
본 3D 모델은 다음과 같이 구조를 분리하여 설계하였다.
전면 프레임 (monitor_top, monitor_bottom)
디스플레이를 고정하고 외관을 형성하는 프레임 구조
후면 케이스 
라즈베리파이 및 전원부를 수용하는 후면 커버 구조
지지대 (monitor leg)
책상 위 거치를 위한 경사 구조의 스탠드
각 부품은 개별 출력이 가능하도록 STL 파일로 분리되어 있다.

-부품 배치 설명
디스플레이는 전면 프레임 내부에 삽입되는 구조로 설계하였다.
라즈베리파이는 후면 케이스 내부에 고정되는 형태로 배치하였다.
HDMI 및 전원 케이블은 측면 개구부를 통해 외부로 인출 가능하도록 고려하였다.
전원부는 후면 커버 내부 여유 공간에 배치할 수 있도록 공간을 확보하였다.
실제 하드웨어를 장착하지 않더라도 가상 배치가 설명 가능한 수준으로 설계하였다.
